# SCSS для kidswedding-inline

Исходник стилей: `kidswedding-inline.scss`. После правок соберите CSS:

```bash
npm run build:css
```

Или включите автопересборку:

```bash
npm run watch:css
```

В начале файла заданы переменные (`$color-bg`, `$color-accent`, breakpoints) — их можно использовать при рефакторинге.
